# 手机端千问模型训练指南

## 📱 可用的千问小模型

| 模型 | 大小 | 手机要求 | 推荐度 |
|------|------|---------|--------|
| Qwen2-0.5B | 0.5B | 2GB内存 | ⭐⭐⭐ 最小 |
| Qwen2-1.5B | 1.5B | 4GB内存 | ⭐⭐⭐⭐ 推荐 |
| Qwen2-7B | 7B | 8GB+内存 | ⭐⭐ 需要高端机 |

## 🚀 方案一：直接在手机上运行（推荐）

### Android 手机

1. **安装 Termux**
   - 下载：https://termux.com
   - 或从 F-Droid 安装

2. **安装依赖**
   ```bash
   pkg install python
   pkg install git
   pip install torch transformers
   ```

3. **下载模型**
   ```bash
   # 使用 huggingface 镜像
   export HF_ENDPOINT=https://hf-mirror.com
   
   # 下载 Qwen2-1.5B-Instruct
   huggingface-cli download Qwen/Qwen2-1.5B-Instruct --local-dir ./qwen-1.5b
   ```

4. **运行推理**
   ```python
   from transformers import AutoModelForCausalLM, AutoTokenizer
   
   model = AutoModelForCausalLM.from_pretrained("./qwen-1.5b")
   tokenizer = AutoTokenizer.from_pretrained("./qwen-1.5b")
   
   # 对话
   input_text = "我今天好累"
   inputs = tokenizer(input_text, return_tensors="pt")
   outputs = model.generate(**inputs, max_length=100)
   print(tokenizer.decode(outputs[0]))
   ```

### 使用 MLChat（更简单）

1. **下载 MLChat**
   - GitHub: https://github.com/nicholasxuusc/MLChat
   - 或搜索 "MLChat" 下载 APK

2. **下载模型**
   - 在 App 内直接下载 Qwen2-1.5B

3. **开始对话**
   - 无需配置，开箱即用

## 🔧 方案二：电脑训练 + 手机运行

### 步骤

1. **在电脑上训练**
   ```bash
   # 使用我们的训练脚本
   python 恋爱训练数据/训练脚本/train_lora.py
   ```

2. **导出模型**
   ```bash
   # 合并 LoRA 权重
   python -c "
   from peft import PeftModel
   from transformers import AutoModelForCausalLM
   
   base = AutoModelForCausalLM.from_pretrained('Qwen/Qwen2-1.5B-Instruct')
   model = PeftModel.from_pretrained(base, './output/love-agent-lora')
   model = model.merge_and_unload()
   model.save_pretrained('./love-agent-1.5b')
   "
   ```

3. **转换格式（可选）**
   ```bash
   # 转换为 GGUF 格式（更小更快）
   pip install llama-cpp-python
   
   # 或使用 llama.cpp
   git clone https://github.com/ggerganov/llama.cpp
   python llama.cpp/convert.py ./love-agent-1.5b --outtype q4_k_m
   ```

4. **传到手机**
   - 通过数据线传输
   - 或上传到网盘下载

## 📦 方案三：使用 GGUF 格式（最省资源）

### 下载预转换的模型

```bash
# Qwen2-1.5B GGUF 版本
wget https://huggingface.co/Qwen/Qwen2-1.5B-Instruct-GGUF/resolve/main/qwen2-1_5b-instruct-q4_k_m.gguf
```

### 在手机上运行

1. **安装 MLC LLM**
   - 下载：https://github.com/mlc-ai/mlc-llm
   - 支持 Android/iOS

2. **加载模型**
   - 将 GGUF 文件放到手机
   - 用 MLC LLM 加载

## 🎯 推荐方案

| 你的情况 | 推荐方案 |
|---------|---------|
| 想快速体验 | MLChat + 预训练模型 |
| 想自己训练 | 电脑训练 + 手机运行 |
| 有编程基础 | Termux + Python |
| 追求性能 | GGUF 格式 + MLC LLM |

## ⚠️ 注意事项

1. **内存要求**
   - 0.5B: 至少 2GB
   - 1.5B: 至少 4GB
   - 7B: 至少 8GB

2. **电量消耗**
   - 运行模型会消耗电量
   - 建议插电使用

3. **发热问题**
   - 长时间运行会发热
   - 注意散热

4. **存储空间**
   - 模型文件较大
   - 确保有足够空间
