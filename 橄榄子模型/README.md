# 橄榄子 Olive - 恋爱实战AI教练

## 简介

橄榄子是一个专注于恋爱实战的AI教练，通过自我进化和对抗训练不断提升能力。

## 特点

- **恋爱实战专家**: 不只是聊天，给具体可执行的建议
- **场景化训练**: 模拟真实恋爱场景，帮助练习
- **心理学支撑**: 每个建议都有心理学原理支撑
- **自我进化**: 通过对抗训练自动生成数据，持续提升

## 文件结构

```
橄榄子模型/
├── olive_model.py      # 主模型（自我进化系统）
├── battle_train.py     # 对抗训练脚本
├── volcano_deploy.py   # 火山平台部署
├── README.md           # 说明文档
└── 进化数据/           # 自我进化生成的数据
```

## 快速开始

### 1. 本地训练

```bash
python olive_model.py
```

### 2. 对抗训练

```bash
python battle_train.py
```

### 3. 火山平台部署

```bash
python volcano_deploy.py
# 然后上传到火山平台
```
