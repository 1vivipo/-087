# 恋爱追女生智能体 - 完整功能与设计文档

## 一、项目概述

### 1.1 项目名称
恋爱追女生智能体（Love Agent）

### 1.2 项目目标
构建一个能够自主学习、自主练习、持续进化的恋爱技巧智能体系统，通过大规模对抗训练，最终达到即使"奇丑无比又一文不名"也能靠技术拿下60%以上女生的目标。

### 1.3 核心理念
- **自主学习**：无需人工干预，自动搜索学习恋爱知识
- **自主练习**：与AI女生角色进行真实聊天博弈
- **群体进化**：20个AI角色互相讨论、共同进步
- **对抗训练**：我方AI vs 敌方海王AI，高强度博弈

## 三、核心模块详解

### 3.1 学习引擎 (Learning Engine)

#### 功能描述
自动搜索和学习恋爱技巧知识，持续扩充知识库。

#### 运行机制
- 每30秒自动学习一个新主题
- 根据预设主题列表随机选择学习方向
- 学习内容自动存入知识库

#### 学习主题
```
吸引力建立、聊天技巧、冷读术、推拉技巧、情绪价值、框架控制
肢体进挪、时间压缩、服从性测试、价值展示、社交认证、预选效应
神秘感、稀缺性、社交圈建设、朋友圈展示、约会技巧、表白时机
关系升级、长期维护、挽回技巧、断联复联、二次吸引、废物测试应对
高难度攻略、海王反制、防御突破、心理战
```

---

### 3.2 对抗引擎 (Battle Engine)

#### 功能描述
与100个不同类型的AI女生角色进行恋爱博弈练习。

#### 角色设计

##### 难度分布
| 难度 | 数量 | 类型 |
|------|------|------|
| 简单(1-3) | 23个 | 邻家女孩、活泼少女等 |
| 中等(4-6) | 25个 | 温柔姐姐、文艺女神等 |
| 困难(7-9) | 42个 | 冰山美人、高冷总裁等 |
| 海王(10) | 10个 | 顶级海王、情感猎手等 |

##### 性格类型
- 冰山美人：冷淡、高傲、难以接近
- 傲娇公主：嘴硬心软、爱面子、别扭
- 温柔姐姐：体贴、成熟、包容
- 活泼少女：开朗、直接、热情
- 高冷总裁：强势、独立、理性
- 文艺女神：敏感、浪漫、理想主义
- 绿茶婊：装纯、心机、利用
- 女汉子：豪爽、直接、不拘小节
- 小公主：娇气、任性、需要宠爱
- 知性女神：聪明、独立、有主见
- 邻家女孩：单纯、善良、好追
- 夜店女王：开放、经验丰富、难定心
- 顶级海王：操控大师、情感收割、备胎无数
- 情感猎手：精准打击、欲擒故纵、价值打压
- 暧昧女王：永远暧昧、从不承诺、吊着你

#### 目标类型
- 速推(quick_score)：快速推进到亲密关系
- 驯化(tame)：让她言听计从
- 攻略(conquer)：让她爱上你
- 恋爱(love)：建立健康恋爱关系
- 结婚(marry)：走向婚姻
- 求财(money)：获取经济利益
- 朋友圈(friendzone)：保持朋友关系
- 炮友(fwb)：保持纯肉体关系
- 干女儿(sugar_baby)：建立干亲关系
- 崇拜(worship)：让她崇拜你

### 3.4 文档系统 (Document System)

#### 功能描述
自动生成和保存各类文档，支持前端查看和下载。

#### 文档类型

| 类型 | 说明 |
|------|------|
| 每日总结 | 每个AI的当日工作总结 |
| 方法论文档 | 每个AI的方法论体系 |
| 技术指南 | 具体技巧的详细说明 |
| 讨论记录 | 所有讨论的完整记录 |
| 博弈记录 | 所有对抗的完整过程 |

#### 导出格式
支持导出为Markdown格式，可直接下载到本地。

## 六、运行机制

### 6.1 自动学习流程
```
启动 → 每30秒 → 选择主题 → AI生成内容 → 存入知识库 → 更新统计
```

### 6.2 自动对抗流程
```
启动 → 每20秒 → 选择女生 → 选择目标 → 6-10轮对话 → 判断胜负 → 保存记录
```

### 6.3 群体运行流程
```
启动 → 每12秒 → 选择AI → 执行动作(学习/对抗/讨论) → 更新统计
```
